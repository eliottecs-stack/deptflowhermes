# Prompt — Premier message après acceptation

Écris un message court et professionnel.

Contraintes :
- 400 caractères maximum.
- Une seule idée.
- Pas de fausse personnalisation.
- Pas de promesse inventée.
- CTA léger.

Structure :
1. contexte réel si disponible ;
2. proposition de valeur ;
3. question simple.
